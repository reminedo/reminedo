# 리마인두 PRD

## 1. 제품 개요

리마인두는 사용자가 특정 시간에 이미지 또는 링크(Youtube, Instagram, Safari 등)를 연결한 알람을 설정하고, 알람이 울렸을 때 바로 행동으로 이어질 수 있게 하는 iOS 알람 앱이다.

핵심 가치는 다음과 같다.

> 알림에서 끝나지 않고, 행동까지 이어지게 한다.

MVP는 서버 없이 동작하며, 알람 데이터와 이미지는 기기 내부에 저장한다.

---

## 2. 핵심 시나리오

## 2.1 다른 앱 사용 중 알람

### 상황

사용자가 유튜브를 보고 있는 중, 10시에 등록한 알람이 울린다.

### 플로우

1. 사용자가 유튜브를 보고 있다.
2. 등록한 시간이 된다.
3. iOS 시스템 알림이 표시된다.
4. 알림에 액션이 표시된다. (unlock.png)
5. 사용자가 `O`를 누르면 등록한 이미지 또는 링크로 이동한다.
6. 사용자가 `X`를 누르면 알림이 종료되고 기존 화면으로 돌아간다.
7. 사용자가 '다시알림'을 누르면 설정한 다시 알림 시간 후에 알림이 예약된다.

---

## 3.2 잠금화면 상태 알람

### 상황

폰이 잠금 상태일 때 등록한 알람이 울린다.

### 플로우

1. 폰이 잠금 상태다.
2. 등록한 시간이 된다.
3. 잠금화면에 알림이 표시된다.
4. 알림에 액션이 표시된다.(lock.png)
5. 사용자가 `O`를 누르면 잠금 해제 후 앱이 열린다.
6. 이미지 알람이면 앱 안에서 이미지를 크게 보여준다.
7. URL 알람이면 등록된 링크를 연다.
8. 메모 알람이면 메모 화면이 들어가진다.
8. 사용자가 `X`를 누르면 알림이 종료된다.
9. 사용자가 '다시알림'을 누르면 설정한 다시 알림 시간 후에 알림이 에약된다.

---

## 3.3 외부 앱 공유를 통한 알람 생성

### 상황

사용자가 유튜브 영상이나 인스타그램 릴스를 나중에 다시 보고 싶다.

### 플로우

1. 사용자가 YouTube, Instagram, Safari 등에서 공유 버튼을 누른다.
2. iOS 공유 시트에 리마인두가 표시된다.
3. 사용자가 리마인두를 선택한다.
4. Share Extension이 공유된 URL을 수신한다.
5. 리마인두 메인 앱의 알람 생성 화면으로 이동한다.
6. 알람 생성 화면에는 공유된 URL이 미리 입력되어 있다.
7. 사용자는 남은 공간을 채운다.
8. 저장하면 알람이 생성된다.

---

## 3.4 이미지 알람 생성

### 상황

사용자가 사진 앱에 있는 단어장 이미지나 체크리스트 이미지를 특정 시간에 보고 싶다.

### 플로우

1. 사용자가 알람 추가 버튼을 누른다.
2. 콘텐츠 타입에서 `이미지`를 선택한다.
3. 이미지 추가 버튼을 클릭한다.
4. iOS Photos Picker가 열린다.
5. 사용자가 이미지를 선택한다.
6. 앱은 이미지를 앱 내부 저장소에 복사한다.
7. 알람 데이터에는 이미지 파일명을 저장한다.
8. 알람이 울렸을 때 사용자가 `O`를 누르면 해당 이미지를 크게 보여준다.

---

## 4. 알람 세부 탭은 바텀 시트 UI임.

### URL 처리

입력 또는 공유된 URL은 별도 타입으로 분류하지 않고 원본 URL 문자열 그대로 저장한다. 알림에서 `O`를 선택하면 저장된 URL을 iOS URL open 방식으로 실행한다.

---




## 5.5 이미지 저장 

### 요구사항 (괜찮은 방식인지 검토필요)

1. 사용자가 선택한 이미지는 앱 내부 저장소에 복사한다.
2. 알람 데이터에는 이미지 파일명만 저장한다.
3. 원본 사진 앱의 이미지를 계속 참조하지 않는다.
4. 원본 사진이 삭제되어도 앱 내부 이미지는 유지된다.
5. 앱 삭제 시 저장된 이미지는 삭제된다.

### 저장 예시

```text
/Documents/reminder-images/{reminderId}.jpg
```

---

## 7. 저장 방식

MVP에서는 서버 없이 로컬 저장만 사용한다.

1. 앱 종료 후 유지
2. 기기 재부팅 후 유지
3. 앱 업데이트 후 일반적으로 유지
4. 앱 삭제 시 삭제
5. 기기 변경 시 자동 복구 없음


## 8. 권한

## 8.1 알림 권한

### 필요 시점

온보딩에서 요청한다.

---

## 8.2 사진 접근

### 방식

iOS Photos Picker를 사용한다.

### 기준

1. 이미지 등록 시 권한이 없다면 전체 이미지 접근 권한을 요구한다.
2. 사용자가 선택한 이미지만 앱으로 가져온다.
3. 가져온 이미지는 앱 내부 저장소에 복사한다.
